## Technical Stack

- React
- Material-UI
- Chart.js
- Local Storage for data persistence

## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/asp2peaK/RiseUp.git
```

2. Install dependencies:
```bash
cd RiseUp
npm install
```

3. Start the development server:
```bash
npm run dev
```
